atlas

modification value options
justify 0: top-weighted
1: center-weighted
link_weight 1: default (1px)
!1: use weights specified in links file
link_type 0: straight line
1: bezier curve (default)
2: bundled bezier curve
link_color any color hex code
link_gradient 0: solid color
!0: gradient
node_bg_color 0: solid color
!0: gradient
node_r 0-50: dictates radius of node corners
